## 一、MCU资源确认

### 1.1 STM32F030K6T6 可用定时器

**重要**：STM32F030K6T6（x6系列，LQFP-32封装）**没有TIM2**！

> 注：EC11编码器用软件EXTI解码（PA11/PA12），不使用硬件编码器模式

| 定时器 | 类型 | 通道数 | 编码器模式 | 本项目用途 |
|--------|------|--------|-----------|-----------|
| TIM1 | 高级定时器 | 4CH+互补 | 不支持 | 备用 |
| **TIM3** | 通用定时器 | 4CH | **支持** | **1ms系统定时器** |
| TIM14 | 基本定时器 | 1CH | 不支持 | 未使用 |
| TIM16 | 基本定时器 | 1CH+互补 | 不支持 | 备用 |
| **TIM17** | 基本定时器 | 1CH+互补 | 不支持 | **PWM加热器** |

> 注意：STM32F030x4/x6 没有 TIM2、TIM15；STM32F030x8 有 TIM15；STM32F030xC 有 TIM2+TIM15

### 1.2 定时器实际方案

| 功能 | 方案 | 原因 |
|------|------|------|
| EC11编码器 | 软件EXTI (PA11/PA12) | PA11/PA12无定时器通道映射 |
| PWM加热器 | TIM17 CH1 (PA7) | TIM3被1ms定时器占用 |
| 1ms定时器 | TIM3 | TIM14配置有问题，TIM3替代 |

### 1.3 PA7 引脚复用验证

PA7 的定时器复用功能：

| AF编号 | 功能 | 可用性 |
|--------|------|--------|
| AF1 | TIM1_CH1N | 可用（互补通道，需MOE使能） |
| AF2 | TIM3_CH2 | 不可用（TIM3已用于编码器） |
| AF4 | TIM14_CH1 | 不可用（TIM14已用于1ms定时） |
| **AF5** | **TIM17_CH1** | **可用（推荐）** |

## 五、电源链路

```
DC插座(DC1) → 保险丝(U5, 10A/125V) → VIN
                                         ↓
                                   TPS54202(U2) DC-DC降压
                                   ├─ EN: R3(22kΩ)上拉到VIN
                                   ├─ BOOT: C12(56pF)
                                   ├─ SW: L1(10uH) → 3V3输出
                                   │              ├─ C10(22uF) → GND
                                   │              └─ C11(22uF) → GND
                                   └─ FB: R5(100kΩ)分压反馈
                                   输入: C7(100uF)

3V3供电: STM32F030(U1), MAX6675(U3), OLED(H1), SWD(H4)
VIN供电: EC11编码器(H3), 加热棒驱动(Q1→P1)
```

## 六、关键电路调试要点

### 6.1 时钟系统

| 问题 | 现象 | 原因 | 解决方案 |
|------|------|------|---------|
| HSE不起振 | 程序卡在Error_Handler | 晶振负载电容不匹配 | C1/C2换30pF(当前22pF)，或改用HSI |
| HSI精度不足 | 串口波特率偏差 | HSI ±1%精度 | PID恒温对时钟精度要求不高，HSI够用 |

**当前方案**：HSI 8MHz → PLL×6 → 48MHz

### 6.2 MAX6675温度读取

| 检查项 | 方法 | 预期结果 |
|--------|------|---------|
| GPIO初始化 | 确认CubeMX已配置PA4/PA5/PA6 | MX_GPIO_Init()中有对应代码 |
| CS拉低后读数 | 用示波器看PA5/PA6波形 | 16个SCK脉冲，MISO有数据 |
| 温度读数为0 | 检查CS/SCK初始电平 | CS=High, SCK=Low(读前) |
| 热电偶断线 | 拔掉热电偶 | D2位=1，temp_thermocouple_ok()返回0 |

### 6.3 EC11编码器

| 检查项 | 方法 | 预期结果 |
|--------|------|---------|
| EXTI中断 | PA11/PA12配置为Rising+Falling EXTI | 旋转时触发中断 |
| 旋转方向 | 顺时针旋转A相 | enc_delta > 0 |
| 按键消抖 | PB0在TIM3(1ms)中轮询 | 20ms消抖,无重复触发 |

### 6.4 PWM加热器

| 检查项 | 方法 | 预期结果 |
|--------|------|---------|
| TIM17 PWM输出 | 示波器看PA7波形 | 1kHz方波 |
| 占空比控制 | 修改CCR值 | 波形宽度变化 |
| MOSFET导通 | PA7高电平 | Q1导通，P1端子有电压 |
| 安全关断 | R2(100kΩ)下拉 | PA7无输出时Q1截止 |

> **安全警告**：首次测试PWM必须低占空比(≤10%)，确保加热器散热良好！

### 6.5 OLED显示

| 检查项 | 方法 | 预期结果 |
|--------|------|---------|
| I2C通信 | 逻辑分析仪看PB6/PB7 | 起始位→0x78→数据 |
| SSD1306地址 | 发送0x78 | 收到ACK |
| 显示花屏 | 检查初始化序列 | 25条命令完整发送 |
| Open-Drain | 确认PB6/PB7为开漏输出 | 需外部上拉或OLED模块自带 |

## 七、CubeMX配置清单（修正版）

### 7.1 SYS
- Debug: `Serial Wire`
- Timebase Source: `SysTick`

### 7.2 RCC
- High Speed Clock: `HSI`（HSE不起振，改用HSI）
- **不要选Crystal/Ceramic Resonator**，否则生成代码会卡在Error_Handler

### 7.3 时钟配置
```
HSI(8MHz) → PLL Source: HSI
          → PREDIV: /1
          → PLLMUL: ×6
          → System Clock: PLLCLK
          → HCLK = 48MHz
          → APB1 = 48MHz
```

### 7.4 USART1
- Mode: `Asynchronous`
- Baud Rate: `115200`
- TX: PA9, RX: PA10

### 7.5 TIM3（1ms系统定时器）
- Clock Source: `Internal Clock`
- Counter Period: `47999`
- Prescaler: `0`
- → 48MHz / 48000 = 1kHz = 1ms中断
- NVIC: 使能 `TIM3 global interrupt`, 优先级0

### 7.6 TIM17（PWM加热器）— 修正！
- Channel1: `PWM Generation CH1`
- Counter Period (ARR): `999`
- Prescaler (PSC): `47`
- Pulse (CCR): `0`
- → PWM频率 = 48MHz / (47+1) / (999+1) = 1kHz
- 引脚: PA7(CH1)，需手动确认AF5映射

### 7.7 TIM14（未使用）
- 保留默认配置，未启用

### 7.8 GPIO

| 引脚 | 模式 | 标签 | 详细设置 |
|------|------|------|---------|
| PA4 | GPIO_Output | MAX6675_CS | Push-Pull, High Speed, 初始High |
| PA5 | GPIO_Output | MAX6675_SCK | Push-Pull, High Speed, 初始High |
| PA6 | GPIO_Input | MAX6675_MISO | NoPull |
| PA8 | GPIO_Output | EC11_C | Open-Drain, 初始Low, 模拟GND |
| PA11 | GPIO_EXTI | EC11_A | Rising+Falling, NoPull, NVIC使能EXTI4_15 |
| PA12 | GPIO_EXTI | EC11_B | Rising+Falling, NoPull, NVIC使能EXTI4_15 |
| PB0 | GPIO_Input | EC11_SW | Pull-Up, 在TIM3中断中轮询消抖 |
| PB6 | GPIO_Output | OLED_SCL | Open-Drain, High Speed, 初始High |
| PB7 | GPIO_Output | OLED_SDA | Open-Drain, High Speed, 初始High |

## 八、常见问题排查流程

### 8.1 程序不运行

```
1. 检查供电 → 3V3是否正常？
2. 检查NRST → 是否被拉低？(R1=10kΩ应上拉到VDD)
3. 检查时钟 → HSE是否配置？→ 改用HSI
4. 检查BOOT0 → 应接GND(从Flash启动)
5. 连接SWD → 能否读取芯片ID？
```

### 8.2 串口无输出

```
1. 检查波特率 → 115200, 8N1
2. 检查MicroLIB → Keil → Options → Target → 勾选Use MicroLIB
3. 检查fputc重定向 → 是否在USER CODE BEGIN 0区域
4. 检查连线 → TTL_RX接PA9, TTL_TX接PA10, GND接GND
5. 检查时钟 → SystemClock_Config是否正确(48MHz)
```

### 8.3 GPIO不工作

```
1. 必须在CubeMX中配置GPIO → 手动写GPIO初始化代码无效！
2. 确认MX_GPIO_Init()被调用
3. 检查引脚是否被其他外设复用
4. 检查GPIO时钟是否使能
```

### 8.4 定时器不工作

```
1. 确认HAL_TIM_MODULE_ENABLED已启用(stm32f0xx_hal_conf.h)
2. 确认MX_TIMx_Init()被调用
3. 编码器模式: 需调用HAL_TIM_Encoder_Start()
4. PWM模式: 需调用HAL_TIM_PWM_Start()
5. 中断模式: 需调用HAL_TIM_Base_Start_IT()
6. TIM1/TIM17: 需使能MOE(主输出使能)
```
