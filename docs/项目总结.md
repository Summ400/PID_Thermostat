## 一、项目概述

基于STM32F030K6T6的PID恒温控制系统，通过K型热电偶读取温度，增量式PID算法计算加热功率，PWM驱动MOSFET控制加热棒，EC11编码器设定目标温度，OLED实时显示。

### 功能

- 实时温度读取（MAX6675 + K型热电偶）
- 编码器设定目标温度（0~120°C，每格1°C）
- 增量式PID自动控温
- PWM驱动加热棒
- OLED显示当前温度和目标温度
- 安全保护：热电偶断线关断、超温120°C关断
- 串口调试输出（115200bps）

### 3.2 模块依赖关系

```
main.c
  ├── temp_reader ──→ soft_spi ──→ GPIO (PA4/PA5/PA6)
  ├── encoder ──→ EXTI (PA11/PA12) + TIM3轮询 (PB0)
  ├── oled_display ──→ soft_i2c ──→ GPIO (PB6/PB7)
  │                  └── oled_font
  ├── pid_control (纯算法，无硬件依赖)
  └── heater_ctrl ──→ TIM17 PWM (PA7)
```

### 3.3 主循环流程

```
初始化:
  HAL_Init → SystemClock → GPIO → USART → TIM3 → TIM17
  enc_init → temp_init → i2c_init → oled_init → heater_init
  oled画静态标签 → pid_init → pid_setpoint(30°C)

主循环 (500ms):
  1. temp_read_all() → 获取温度+热电偶状态
  2. enc_get_delta() → 编码器调节目标温度
  3. heater_is_safe()?
     是 → pid_compute() → heater_set_duty()
     否 → heater_off() + pid_reset()
  4. 串口打印状态
  5. OLED刷新数字
```

## 四、开发历程

### 阶段0：CubeMX工程 + 串口

- STM32CubeMX配置时钟、GPIO、USART1
- printf重定向到UART1（需启用MicroLIB）
- 验证：串口输出 "Hello PID Thermostat!"

### 阶段1：MAX6675温度读取

- 软件SPI读取MAX6675（PA4/PA5/PA6）
- 16位原始数据解析：D14~D3为12位温度值（0.25°C/LSB），D2为热电偶断线标志
- 验证：串口输出室温，手握热电偶温度上升

### 阶段2：EC11编码器

- PA11/PA12 EXTI双边沿触发检测旋转方向
- PB0在TIM3(1ms)中断中轮询消抖+长按检测
- PA8开漏输出低电平模拟EC11公共端接地

**踩坑：**
- EC11引脚顺序是A-C-B，不是A-B-C，接错后B相始终为0
- EXTI方式对机械编码器有抖动问题，但正确接线后可正常工作
- PA11/PA12在GPIO_Input模式下必须Pull-up，EXTI模式NoPull即可

### 阶段3：SSD1306 OLED显示

- 软件I2C驱动SSD1306（PB6/PB7，地址0x78）
- 8×16字体，0.91寸128×32分辨率，2行显示
- 静态标签只画一次，数字每周期刷新，避免闪烁

### 阶段4：PID控制 + PWM加热

- 增量式PID算法，输出0~65535对应PWM占空比
- TIM17_CH1输出PWM到PA7驱动AOD4184A
- 安全保护：热电偶断线/超温120°C强制关断+PID重置
- temp_read_all()合并为一次SPI读取，避免连续读取导致温度跳动

**踩坑：**
- int16_t装不下65535，PID输出改为int32_t
- 不安全时必须pid_reset()，否则恢复后输出突变
- 增量式PID参数不能直接抄位置式的，Kp/Ki含义不同

## 五、PID参数说明

### 5.1 增量式PID公式

```
Δu = Kp × (e(k) - e(k-1))           ← 比例项：误差变化趋势
   + Ki × e(k)                       ← 积分项：消除稳态误差（主力项）
   + Kd × (e(k) - 2e(k-1) + e(k-2)) ← 微分项：抑制超调

新输出 = 旧输出 + Δu
```

### 5.2 当前参数

```c
pid_init(&heater_pid, 80.0f, 15.0f, 10.0f, HEATER_MAX_DUTY);
```

| 参数 | 值 | 作用 |
|------|----|------|
| Kp | 80 | 误差变化时快速调整 |
| Ki | 15 | 主力项，根据当前误差大小决定加热力度 |
| Kd | 10 | 抑制超调 |

### 5.3 增量式 vs 位置式参数区别

增量式的Ki对应位置式的Kp（主力），增量式的Kp对应位置式的Kd（阻尼），不能直接套用位置式参数。

### 5.4 调参方向

| 现象 | 调整 |
|------|------|
| 升温太慢 | 加大Ki |
| 温度冲过头（超调） | 减小Ki，加大Kd |
| 到不了目标温度 | 加大Ki |
| 温度来回振荡 | 减小Ki，减小Kp |
| 稳定后小幅抖动 | 减小Kp |

### 5.5 参考参数来源

Marlin 3D打印机固件（位置式PID）：
- 喷头：Kp=22.20, Ki=1.08, Kd=114.00
- 热床：Kp=46.73, Ki=0.84, Kd=652.00

换算到增量式后作为本项目初始参数。

## 七、安全机制

| 条件 | 动作 |
|------|------|
| 热电偶断线（MAX6675 D2=1） | heater_off() + pid_reset() |
| 温度 ≥ 120°C | heater_off() + pid_reset() |
| 正常状态 | PID自动控制PWM |

## 八、串口协议

波特率115200，8N1，每500ms输出：

```
PID OUT: 30000 / 65535           ← PWM占空比
Temp: 35.2 C | Target: 50.0 C | TC: OK    ← 状态
Target: 32.0 C                   ← 编码器调节时输出
HEATER OFF (TC:OPEN! Temp:35.0)  ← 安全关断时输出
BTN SHORT PRESS / BTN LONG PRESS ← 按键事件
```

## 九、已知问题

1. HSE外部晶振不起振（负载电容22pF偏小，应换30pF），当前用HSI
2. TIM14在CubeMX中没有Internal Clock选项，用TIM3代替
3. TIM17 PWM频率约732Hz（ARR=65535），可改为1kHz
4. PID参数需实测调整，当前为参考值
